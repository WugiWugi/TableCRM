# TableCRM Mobile Order — minimal Next.js app

1. Установить зависимости:

npm install

2. Запустить локально:

npm run dev

3. Открыть http://localhost:3000

4. Как тестировать:
- Вставить токен в поле "Токен кассы".
- Выполнить поиск клиента по телефону.
- Выбрать счёт, организацию, склад, тип цен.
- Добавить товар(ы).
- Нажать "Создать" или "Создать и провести".
- В ответе отобразится результат запроса.

5. Деплой:
- Подключить репозиторий в Vercel, стандартный билд `npm run build`.
- URL будет доступен публично. (Не хранить публично секретные токены.)

Примечание: API-руты и формат ответов в проекте обработаны максимально универсально: библиотека `lib/api.js` ожидает стандартную структуру `data/items/contragents` и адаптирует результат. При необходимости подправить конкретные ключи под реальный ответ API.
